# Security Policy

## Reporting a Vulnerability
PulseThread deals with sensitive user data (location and health status).

**DO NOT** open public issues for security vulnerabilities.

If you discover a security issue, please message me directly at: **[linkedin.com/in/akhlakhossainjim]**.

I will acknowledge your message within 48 hours and provide an estimated timeline for the fix. Security researchers who responsibly disclose vulnerabilities will be credited in the Release Notes.
